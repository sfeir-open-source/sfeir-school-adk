# Workshop : Assistant de recherche multi-agents

## Lab

<br>

Construisez un assistant de recherche qui utilise plusieurs patterns d'agents pour collecter, analyser et résumer des informations.

- ✅ Implémenter un pipeline séquentiel

- ✅ (Bonus) Créer un custom agent avec routing intelligent
